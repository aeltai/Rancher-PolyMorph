# s3

Pull and push Rancher backup tarballs via S3.

Configure `s3.bucket`, `region`, and optional `prefix` in `rancher-migrate.yaml`.

## pull

```bash
rancher-migrate s3 pull migrations/source-full.tar.gz -o ./backups/in.tar.gz
```

## push

```bash
rancher-migrate s3 push ./backups/sanitized.tar.gz migrations/out.tar.gz
```

## list

```bash
rancher-migrate s3 list
rancher-migrate s3 list migrations/
```

Supports custom endpoints (MinIO) via `s3.endpoint` in config.
